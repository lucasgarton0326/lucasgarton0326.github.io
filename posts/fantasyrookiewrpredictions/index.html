Fantasy Rookie WR Predictions

football
fantasy
Author

Lucas Garton

Published

May 11, 2023

With the 2023 draft in the review mirror, I decided to look into who would be the next Jamar Chase or Justin Jefferson for the upcoming fantasy season. To do this I used machine learning to help predict what this years class of rookie wide receivers would be expected to score.

To predict there total points for the upcoming season I used multiple different factors from their senior season. To start with I used the total yards, yards per reception,total touchdowns because I thought those three stats summed up what an individual WR can do best. Then next I used simple rating because obviously a player like Zay Flowers,who played at Boston College, isn’t going to play nearly the level of talent that a Jaxon Smith-Njigba does playing at Ohio State. Lastly I used overall and pre-draft grade because I figured normally good receivers get drafted for the most part.

After running through all the data and using a combination of LightGBM and XGBoost to figure out who draft my model tells me to draft… Jalin Hyatt first. I guess my model can not account for the fact that the most bland QB in the league Daniel Jones would be throwing him the ball. Second was super close and can be considered more of a toss up with Quentin Johnston and Jordan Addison separated by only 1.3 predicted fantasy points.

Code 
library(tidyverse)
library(tidymodels)
library(cfbfastR)
library(janitor)
library(bonsai)
library(gt)
set.seed(1234)

wrstats <- read_csv("https://mattwaite.github.io/sportsdatafiles/wrdraftedstatswithratingsfixed.csv") 
currentwrstats <- read_csv("https://mattwaite.github.io/sportsdatafiles/currentwrstats.csv")

wrselected <- wrstats %>%
  select(
    name,
    year,
    college_team,
    nfl_team,
    overall,
    total_yards,
    rating,
    total_ypr,
    pre_draft_grade,
    total_touchdowns,
    FantPt
  ) %>% na.omit()

player_split <- initial_split(wrselected, prop = .8)
player_train <- training(player_split)
player_test <- testing(player_split)

player_recipe <- 
  recipe(FantPt ~ ., data = player_train) %>%
  update_role(name, year, college_team, nfl_team, new_role = "ID")

xg_mod <- boost_tree(
  trees = tune(), 
  learn_rate = tune(),
  tree_depth = tune(), 
  min_n = tune(),
  loss_reduction = tune(), 
  sample_size = tune(), 
  mtry = tune(), 
  ) %>% 
  set_mode("regression") %>% 
  set_engine("xgboost")
lightgbm_mod <- 
  boost_tree() %>%
  set_engine("lightgbm") %>%
  set_mode(mode = "regression")

xg_workflow <- 
  workflow() %>% 
  add_model(xg_mod) %>% 
  add_recipe(player_recipe)
lightgbm_workflow <- 
  workflow() %>% 
  add_model(lightgbm_mod) %>% 
  add_recipe(player_recipe)

xgb_grid <- grid_latin_hypercube(
  trees(),
  tree_depth(),
  min_n(),
  loss_reduction(),
  sample_size = sample_prop(),
  finalize(mtry(), player_train),
  learn_rate()
)
player_folds <- vfold_cv(player_train)
xgb_res <- tune_grid(
  xg_workflow,
  resamples = player_folds,
  grid = xgb_grid,
  control = control_grid(save_pred = TRUE)
)
best_rmse <- select_best(xgb_res, "rmse")
final_xgb <- finalize_workflow(
  xg_workflow,
  best_rmse
)

xg_fit <- 
  final_xgb %>% 
  fit(data = player_train)
lightgbm_fit <- 
  lightgbm_workflow %>% 
  fit(data = player_train)

xgpredict <- 
  xg_fit %>% 
  predict(new_data = player_train) %>%
  bind_cols(player_train) 
lightgbmpredict <- 
  lightgbm_fit %>% 
  predict(new_data = player_train) %>%
  bind_cols(player_train) 

lightgbmpredict <- 
  lightgbm_fit %>% 
  predict(new_data = player_test) %>%
  bind_cols(player_test) 

xgpredict <- 
  xg_fit %>% 
  predict(new_data = player_test) %>%
  bind_cols(player_test) 

currentwrselected <- currentwrstats %>%
  select(
    name,
    year,

    overall,

currentxgpredict <-
